#백준 13144 List of Unique Numbers(G4)
    - 투 포인터 문제
    - 우선 모든 경우의 수만 해도 n^2의 부분 수열의 개수가 나오므로 무식하게 풀 수 없음
    - 이때 연속적인 부분 수열을 판단하기 때문에, 투 포인터를 활용하여 문제를 해결할 수 있음
    - 주의할 점, 최대 수열의 개수는 n * (n + 1) / 2이므로 int형의 범위를 넘기 때문에
      정답을 구할 때, 오버플로우가 나지 않게 자료형을 잘 선언해야 함.
    - 또한, long long = int * int일 경우 int * int에서 int범위를 넘게 되면, 
      int * int = int이므로 long long 자료형에 저장한다고 해서, 값이 올바르게 저장되지 않음
      이때, long long = 1ll * int * int를 통해 앞의 1ll * int로 long long 자료형으로 계산이 되기 때문에 이를 방지할 수 있음
